IIT Dharwad — Real-Time Bus Tracking System

About the Shuttle Service

The IIT Dharwad Shuttle Bus Service provides reliable transportation between the Permanent Campus and key city locations, ensuring smooth daily mobility for students, faculty, and staff.

Schedules follow the official institute timetable. For the latest timings and route details, refer to the official schedule.

Hostel Main Gate NFSU New Bus Stand Jubilee Circle A1 / A2 / CLT

How It Works

An onboard GPS + LTE module pushes live location data to the ThingSpeak IoT cloud. The web app fetches this every 20–60 seconds and overlays the bus position on an interactive map, combined with schedule data so every page stays accurate without a manual refresh.

Arduino Mega SIMCom A7672S LTE GPS ThingSpeak API Leaflet.js HTML / CSS / JS
GPS updates every 60 s (tracking page) and 20 s (stop pages)
Timetable auto-switches between weekday and weekend / holiday schedules
Proximity alerts notify you when the bus is ~10 min from your chosen stop
Status shows Live or Parked based on GPS staleness
