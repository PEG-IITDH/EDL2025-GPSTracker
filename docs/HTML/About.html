IIT Dharwad — Real-Time Bus Tracking System

About the Shuttle Service

The IIT Dharwad Shuttle Bus Service provides reliable transportation between the Transit Campus, the Permanent Campus, and key city locations, ensuring smooth daily mobility for students, faculty, and staff.

Schedules follow the official institute timetable. For the latest timings and route details, refer to the official schedule.

How It Works

The tracking system uses an onboard GPS + LTE module to push location data to ThingSpeak IoT cloud in real time. The web application fetches this data every 20–60 seconds and overlays the bus position on an interactive map, alongside live schedule data so every page stays accurate without a refresh.

Arduino Mega SIMCom A7672S LTE GPS ThingSpeak API Leaflet.js HTML / CSS / JS
GPS updates every 60 s (tracking page) and 20 s (stop pages)
Timetable automatically switches between weekday and weekend / holiday schedules
Proximity alerts notify you when the bus is ~10 min from your chosen stop
Status shows Live or Parked based on GPS staleness
